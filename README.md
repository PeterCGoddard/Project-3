Project Overview: React App with Todo List, Contact Page, and navbar to switch between pages

Functionality: Navbar links to home page, Todo List, and Contact Page. In Todo List, user can create a to do list, mark items as complete, and delete items.
In Contact Form, user can input first name, last name, email, and comments.

Technologies: React, .js, .css, github

Ideas for Future Improvement: Add colored backgrounds, add print option for Todo List, add donations page.
